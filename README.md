## Vacation Manager
휴가 관리를 위한 서비스 개발

## 목차
1. [개발 환경](#1-개발-환경)
2. [개요](#2-개요)
3. [주요 기능](#3-주요-기능)

## 1. 개발 환경
- Spring Boot 2.4.3
- JDK 11

## 2. 개요
연차휴가 관리하는 서비스가 없는 회사에서 연차 관리를 하기 위한 프로젝트

개발 완료 후 실제 사용을 목표로 한다.

## 3. 주요 기능
- 사용자는 회원가입을 하여 본인의 계정으로 로그인을 한다.
- 사용자는 휴가신청을 올릴때 결재받을 결재선을 지정하여 휴가정보와 함께 신청을 한다.
- 결재자들은 결재 단계에 맞춰 자기 차례가 오면 결재를 진행하고 최종 결재가 끝나면 휴가 처리 담당자가 처리를 하고 완료를 한다.
