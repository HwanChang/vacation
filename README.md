## 1. 개발 환경
- Spring Boot 2.4.3
- JDK 11

## 2. 개요
연차휴가 관리하는 서비스가 없는 회사에서 연차 관리를 하기 위한 프로젝트

개발 완료 후 실제 사용을 목표로 한다.

## 3. 주요 기능
- 사용자는 회원가입을 하여 본인의 계정으로 로그인을 한다.
- 사용자는 휴가신청을 올릴때 결재받을 결재선을 지정하여 휴가정보와 함께 신청을 한다.
- 결재자들은 결재 단계에 맞춰 자기 차례가 오면 결재를 진행하고 최종 결재가 끝나면 휴가 처리 담당자가 처리를 하고 완료를 한다.
- 사용자는 한 번 로그인 완료 후 로그아웃을 하지 않으면 자동으로 로그인 상태를 유지한다.

## 4. 서비스 설명
1. 회원가입을 통해 사용자 계정 생성
    - 회원가입 후 휴가 신청 메뉴에서 휴가 신청 진행
    - 다른 계정을 결재자로 지정 가능하고 자가 결재 지정도 가능
    

2. 신청을 올리고 나면 휴가 신청 현황 메뉴에서 신청한 신청서 리스트 확인


3. 본인에게 올라온 결재는 결재 메뉴에서 결재 버튼 통해서 결재
    - 결재 현황 버튼을 눌러 해당 신청서의 현재 결재 현황을 확인 가능
    

4. 처리자로 로그인 하면 처리 메뉴가 활성화 되고 해당 메뉴에서 처리버튼 통해 신청서 최종 처리
    - 결재가 완료된 신청서 처리는 처리 권한을 가진 계정만 가능 (아래 처리자 계정으로 테스트 가능)
